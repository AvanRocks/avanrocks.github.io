String Art

A program that mimics photographs by drawing many chords in a circle.

One day, I watched this Youtube video where a guy shows how he made a machine that winds string around a bunch of pegs arranged in a circle and the string ends up recreating a photograph of an eye.

Go watch the video, it's really cool.

So it turns out that this type of art is a thing, and it's called 'string art'. Some people make this sort of art by hand, by winding a string around the pegs manually. There are two hard parts about making string art.

  • One is deciding which pegs to wind the string around and it what order.
  • The other is the labor of winding the string thousands of times.

The guy in the video made both of these steps easier. He wrote code to decide how to wind the string and built a machine to do it.

I thought this idea was cool so I wanted to try making my own string art. Since I lack the hardware skills to make a machine, I decided to simulate making string art by drawing it on the screen. You can check out the code on my Github. There's a README there that explains how to build the program and use it yourself.
